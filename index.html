About Me

I am a developer from India with a passion for building beautiful and functional web applications. My journey started with a fascination for how code can transform a blank screen into a powerful tool that impacts people's lives.

My Philosophy

I believe that technology should be invisible. The best software doesn't scream for attention; it quietly empowers the user to do more. Following the principles of the Human Interface Guidelines, I aim to create software that feels native to the user's workflow.

Whether it is optimizing a JavaScript engine for faster load times or refining the kerning in a typography-heavy layout, I pay attention to the details that others might overlook.

Technical Background

My expertise lies in front-end development, specifically in creating fluid interfaces with modern frameworks. I have spent countless hours mastering the art of CSS architecture and exploring the depths of high-performance rendering. I am deeply committed to the idea of "Web Performance as a Feature," ensuring that every millisecond is accounted for.

In addition to web technologies, I have a strong foundation in computer science fundamentals, which allows me to approach problems with a logical and structured mindset.
